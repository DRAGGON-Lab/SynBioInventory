# draggon_inventory_sbol

Domain-focused SBOL helper package for SynBioInventory.

## What this package provides

- Inventory and storage schema classes
- Factory methods for MVP implementation kinds
- Placement validation against top-level freezer/fridge policies
- Minimal RDF/XML serialization helper for backend submission scaffolding

## Status

This is an MVP scaffold intended for iterative hardening against live SynBioHub behavior.
